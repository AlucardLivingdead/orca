Ranked Orca Bugs, RFEs, and Other Tasks

The following table represents a first pass at ranking and prioritizing the bugs, RFE's, and other known desires for Orca as of 04-Feb-2007. Numerical rankings are merely a floating point value, where a lower number is more important than a higher number. As things are moved around, we'll just adjust the floating point numbers to match. That is, the only relevance in the number is to give a ranking. The relative closeness or distance of the ranking value of consecutive items is not meaningful.

The target entries followed with an exclamation point (e.g., "GNOME 2.18!") mean we view it as critical to the task done by that target. Target entries followed by nothing (e.g., "GNOME 2.18") mean we highly desire to reach that target. Target entries followed by a question mark (e.g., "GNOME 2.18?") mean we would like to reach that target.

Following this table is another table of unranked bugs and RFE's. They have yet to be inserted into the first table.
